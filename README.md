# Common Sense Machines (csm)

Common Sense Machines (CSM) builds 3D world models with image-to-3D and 3D world simulation APIs aimed at games, robotics, and synthetic data. The Cube product family released a single-click 3D model pipeline with mesh-parts-based topology in early 2026.
